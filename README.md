# 图书推荐分析系统 python609

本项目是一款基于 Python Django 框架开发的图书推荐分析系统。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 开发工具：PyCharm（推荐）

## 功能模块

### 系统角色

1. 普通用户：包含登录、注册等功能。
2. 管理员：负责系统管理。

### 功能说明

1. 可视化图表展示：支持各类图表的单独生成。
2. 数据展示：展示图书数据。
3. 推荐功能：为用户推荐合适的图书。
4. 可视化大屏：直观展示系统数据。
5. 超级管理员后台：

   - 使用 Django-admin 构建后台。
   - 代码结构清晰简单，易于二次开发与功能定制。
